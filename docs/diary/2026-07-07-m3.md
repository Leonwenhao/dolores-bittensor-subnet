## Scope

Completed the M3 offline demo floor: in-process miners, Docker-backed validator

## Changes

- Added `src/dolores_subnet/epoch.py` with full offline epoch orchestration:
  collect submissions, validate, append evidence, top-k scores, EMA state,
  normalized weights, and `weights_epoch_<E>.json`.
- Added offline miner personas in `neurons/miner.py`:
  `honest`, `duplicate_spammer`, `invalid`, and `lazy`.
- Added `scripts/local_epoch.py` for the M3 offline epoch gate.
- Added `scripts/report.py` for leaderboard rendering and `--replay-check`.
- Added scoring helpers for top-k epoch scores, EMA updates, active-hotkey
  weight normalization, and deterministic artifact normalization.
- Added M3 tests: `tests/test_epoch_offline.py` and
  `tests/test_scoring_weights.py`.
- Deleted old scaffold files `scripts/local_loop.py` and
  `src/dolores_subnet/dolores_bridge.py`.
- Updated README smoke commands to the M3 offline loop.
