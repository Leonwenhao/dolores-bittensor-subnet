no GitHub push was attempted, and no paid provider calls were made.

## Implementation

Added a small M4 wire layer:

- `src/dolores_subnet/wire.py`: `DoloresTaskSynapse`, endpoint parsing, Dendrite
  query helper, and wire miner wrapper.
- `neurons/miner.py`: `--mode wire` starts a local Bittensor Axon and serves
  Dolores task submissions from the existing miner personas.
- `neurons/validator.py`: `--mode wire` queries local miner axons, runs the same
  Docker-backed Dolores validation path, writes archive records, and emits the
  fallback weight artifact.
- `src/dolores_subnet/bridge.py`: initially recorded unreachable wire endpoints
  as `infra_error`; this was corrected in the M4 hardening pass.
- `tests/test_wire.py`: coverage for the synapse and endpoint parser.

## Happy Path Run

Started two local miner axons:

```bash
.venv/bin/python neurons/miner.py --mode wire --persona honest --quota 2 \
  --seed 201 --port 8091 --wallet.name dolores-test --wallet.hotkey miner-0

.venv/bin/python neurons/miner.py --mode wire --persona duplicate_spammer \
  --quota 2 --seed 202 --port 8092 --wallet.name dolores-test \
  --wallet.hotkey miner-1
```

Validator command:

```bash
.venv/bin/python neurons/validator.py --mode wire \
  --miner-endpoints 127.0.0.1:8091:5FHE1ZpqMVWS3wyjbTmqDP4obyT1UhJ5fmzJWkf8noAVxkEA,127.0.0.1:8092:5DhPKfTfbN5nwTx3riEQz5TPZH94PV8GLV8oqFMyFG61ZkQg \

  --wallet.name dolores-test --wallet.hotkey validator \
  --timeout 45
```

- Duplicate-spammer miner accepted: 0/2.
- Weights: honest `1.0`, duplicate-spammer `0.0`.

## Kill Test

```bash

  --miner-endpoints 127.0.0.1:8091:5FHE1ZpqMVWS3wyjbTmqDP4obyT1UhJ5fmzJWkf8noAVxkEA,127.0.0.1:8092:5DhPKfTfbN5nwTx3riEQz5TPZH94PV8GLV8oqFMyFG61ZkQg \
```

Result:

- Honest miner accepted: 2/2.

- Validator did not abort.
- Weights: reachable honest miner `1.0`, unreachable miner `0.0`.
- Report: `work/m4_wire_kill/subnet_archive/epochs/epoch_1/report_epoch_1.md`.
